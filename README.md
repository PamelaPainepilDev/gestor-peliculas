## Gestor de Películas

Aplicación web desarrollada con Node.js, Express y Hadlebars que permite gestionar una colección de películas.

## Objetivos
- Listar películas registradas

- Agregar nuevas películas con
datos y fotos (afiche)

- Persistir los datos en un      archivo JSON para que no se pierdan al reiniciar el servidor

- Subir imágenes con multer

## Tecnologías usadas
- Node.js
- Express
- Express-Handlebars
- Multer
- HTML / CSS
